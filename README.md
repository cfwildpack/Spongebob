# 🧽 SpongeBob Fun Project

A multi-language demo project inspired by SpongeBob SquarePants.

## Features
- Character simulation (Python)
- Krabby Patty order system (JavaScript)
- Bikini Bottom web page (HTML/CSS)
- Simple API server (Node.js)
- Ocean engine (C++)
- Character model (Java)
- Jellyfish counter (Go)
- Diagnostics tool (Rust)
- Configuration files (JSON & YAML)

## Characters
- SpongeBob
- Patrick
- Squidward
- Mr. Krabs
- Sandy
